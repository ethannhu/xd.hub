## 💡 愿景

通过社群共建的资源共享，致力于：
* **弥补缺失**：填补学院课程培养方案中实践与经验的空白。
* **打破壁垒**：消除信息差，让学术资料与前人经验触手可得。
* **减负增效**：降低重复劳动，助力更高价值的个人素质发展。

---

## 📂 资源概览

项目根据学院/课程结构分类，主要包含以下内容：

### 内容分类
| 类别 | 说明 |
| :--- | :--- |
| **缺失一课** | 必需而学院未教授的工程技能 |
| **实验攻略** | 报告模板、环境配置及避坑提示 |
| **大作业攻略** | 技术栈推荐及项目案例 |
| **课程攻略** | 往年回顾及知识树结构总览 |
| **经验之谈** | 保研、考研、就业及其他分享 |

### 覆盖学院
- [x] **网络与信息安全**
- [x] **空间科学与技术**
- [x] **集成电路**
- [ ] *更多学院持续建设中...*

## 🛠️ 使用指南

### 1. 查找资源
可以直接通过 [GitHub Pages](https://ethannhu.github.io/xd.hub) 浏览网站，或使用仓库目录树进行层级索引。

### 2. 交流提问
若有问题，请优先通过 [Issues](https://github.com/ethannhu/xd.hub/issues) 提交反馈。

### 3. 参与贡献
我们鼓励任何形式的共建：


> **注意：** 请在贡献时隐去**个人敏感隐私信息**，并确保内容**不违反相关保密规定**。

**基于 PR**
1. **Fork** 本项目。
2. 在对应目录下新增或修订内容。
3. 提交 **Pull Request**。

**基于在线社群**
- *建设中！*
